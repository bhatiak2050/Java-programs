Here, all programs are run from Command Prompt.
Syntax: Programs are written directly into .java files.
in cmd we write:

  javac Filename.java

if it compiles successfully it'll give no error or message. Then type

  java Filename 

Sequence of events: First an intro to programming
working of electronics systems, OS, 
how an app behaves in a computer (Hotel example)
Explaination of basic OOPs concepts (Abstraction, Encapsulation, Polymorphism, Inheritance)
 
Then we setup java on the cmd (must add jdk_version/bin to PATH)
make a folder and in it write the java programs above. These programs explain basic OOPS Concepts.
